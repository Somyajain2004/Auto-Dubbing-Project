# Auto-Dubbing-Project

Tech used : 
| Step | Tool |
|------|--------------------------------|
| 1. Speaker Diarization | pyannote.audio |
| 2. Transcription (ASR) | Whisper |
| 3. Translation | Hugging Face NLLB |
| 4. Voice Synthesis (TTS) | Coqui TTS (XTTS) |
| 5. Audio Assembly | pydub |
